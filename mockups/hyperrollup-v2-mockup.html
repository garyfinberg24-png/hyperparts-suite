Cross-Cutting Features (All Views)

NEW

Auto-Refresh

Configurable polling interval (15s–5min). Green pulsing dot in toolbar. Manual refresh button. Pause on hover. Pattern from HyperCharts.

Audience Targeting

Per-item audience targeting via Azure AD group membership. Batch group check API. Fail-open for resilience. Pattern from HyperLinks.

Click Analytics

Track view mode changes, item clicks, preview opens, searches, and exports. Stored in SP list. Pattern from HyperLinks analytics.

"NEW" Badge

Orange pill badge on items modified within N days (configurable). Shows on all layouts. Computed via isNewItem(item, days).

Demo Mode

5 sample data presets (Documents, News, Events, Projects, HR). Toggle bar outside edit mode. Preset selector dropdown. Great for demos!
